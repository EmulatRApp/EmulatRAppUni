Example (Console) execution

A console debugging session on a current build (1024 MB, DS10 268 MHz). Note the cosmetic "ISA table corrupt" message at power-up (the firmware reinitializes the ISA table to defaults), the "system serial number not set" advisory, and the use of e (examine) and d (deposit) to inspect physical memory.
