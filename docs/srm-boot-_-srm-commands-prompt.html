DS10 -- SRM Boot, Commands & SuperIO Controller

The DS10 firmware reference. SRM boot and the >>> command set are covered operationally in the Quick Start and Operating EmulatR chapters; this appendix records the DS10-specific SuperIO analysis -- the part of the platform that is unique reference material rather than general operation.

 

SRM boot and commands (pointers)

Reaching the >>> prompt and the boot sequence are described under Quick Start; the console command catalogue (show config, show device, boot, set, and the rest) is under Operating EmulatR. The notes below assume that context and focus on the SuperIO programming the firmware performs during initialization.

 

SuperIO controller -- word-fold (confirmed)

The Cypress CY82C693 SuperIO is configured through 16-bit writes that fold an index byte and a data byte into one word: the low byte is the index, the high byte is the data. This byte order is locked and confirmed -- the 0x22 / 0x23 power-control writes carry their Table 61 default data bytes, and the configuration splits cleanly into port 0x3F0 (index) / port 0x3F1 (data):

SIO-TRACE W port=0x3F0 val=0x60 [cfg] index 0x60

SIO-TRACE W port=0x3F1 val=0x03 [cfg] data 0x03 (FDD base high)

SIO-TRACE W port=0x3F0 val=0x70 [cfg] index 0x70

SIO-TRACE W port=0x3F1 val=0x06 [cfg] data 0x06 (IRQ6)
