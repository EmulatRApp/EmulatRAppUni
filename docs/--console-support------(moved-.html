Console Support

EmulatR exposes its console as a RAW TCP stream (default localhost:10023) -- there is no Telnet option negotiation, so a raw-capable client is preferred. Any of the following can attach:

• PuTTY (Raw)

• Plink (-raw)

• Tera Term (TCP/IP)

• MobaXterm (TCP)

• netcat (nc) -- handy for scripting and automated testing

• telnet (may connect, but raw is preferred)

 

If EmulatR runs on another machine, replace localhost with that host's name or IP. For development and debugging the recommended client is Plink with verbose logging:

plink -v -raw localhost -P 10023

 

See TCP Console Overview for the port and auto-launch behavior, and Terminal Clients for the step-by-step connection in each program.
