Attributions & Credits

EmulatR stands on a great deal of prior work. This page records the principal sources and acknowledgments.

 

Architecture and firmware heritage

Digital Equipment Corporation (DEC) / Compaq / HP -- the Alpha AXP architecture, the 21264 / 21272 hardware, the SRM console, and the PALcode. The Alpha Architecture Reference Manual and the DEC firmware sources are the authoritative references EmulatR is measured against.

The MILO and apisrm source trees -- reference firmware / PALcode sources consulted during development.

 

Libraries and algorithms

Mark Adler -- the inflate decompression engine (version c10p1) used by the DEC SRM decompressor and mirrored in the host oracle.

Berkeley SoftFloat (John Hauser) -- the IEEE / VAX floating-point reference implementation behind the FP backend.

spdlog -- the logging library.

Qt -- the UI / console framework at the periphery.

 

Tools

Ghidra -- firmware reverse-engineering. PuTTY / plink (Simon Tatham) -- the console terminal client. AXPBox -- consulted as a non-authoritative differential cross-check.

Performance Profiling - Performance Validator https://www.softwareverify.com/product/performance-validator/

Documentation Tooling - Help & Manual https://www.helpandmanual.com/
