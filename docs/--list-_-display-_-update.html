LFU Commands

At the UPD> prompt the utility presents a function menu. ? or Help scrolls it:

 

 ***** Loadable Firmware Update Utility *****

 Function Description

 Display Displays the system's configuration table.

 Exit Done exit LFU (reset).

 List Lists the device, revision, firmware name, and update revision.

 Readme Lists important release information.

 Update Replaces current firmware with loadable data image.

 Verify Compares loadable and hardware images.

 

List

list shows each device with its current revision, firmware filename, and the revision available to update to:

UPD> list

Device Current Revision Filename Update Revision

nt Unknown nt_fw 5.71

srm 7.3-1 srm_fw 7.3-1

 

Display

display prints the system configuration table (the same PCI / ISA inventory shown by show config at the console). See show config and the Example - Performance transcript for the full layout.

 

Update

update <device> replaces the named device's firmware in flash. Updating srm refreshes the SRM console image; an unknown device name is rejected ("not supported under LFU").

UPD> update srm

Confirm update on:

srm [Y/(N)]y

WARNING: updates may take several minutes to complete for each device.

 DO NOT ABORT!

srm Updating to 7.3-1... Verifying 7.3-1... PASSED.

 

When done, exit leaves LFU (a reset) and the firmware continues to the >>> prompt.
