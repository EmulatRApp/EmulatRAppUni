Appendix DS20 / ES40 / ES45

The larger Tsunami-family machines are planned but not yet modeled or validated in EmulatR. They are recorded here so the platform roadmap is explicit; none of them currently reaches the >>> prompt.

 

DS20

Planned. DS20 is not yet modeled or validated. A platform manifest and reference notes will be added once the model exists.

 

ES40

Planned. The platform manifest es40_platform.json is scaffolded, but the platform has not yet been driven to the >>> prompt. To be validated and documented.

 

ES45

Planned. The platform manifest es45_platform.json (EV67 / 21264A) is scaffolded but not yet validated. To be documented once it boots.

 

Note that the compressed firmware oracle work (host_decompressor) was developed against es45_v7_3.exe, so the ES45 firmware image decompresses correctly even though the full platform is not yet driven; what remains is the device and chipset modeling around it.
