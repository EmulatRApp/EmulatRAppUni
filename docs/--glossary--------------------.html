Glossary

Terms and acronyms used throughout this guide.

 

Alpha AXP -- Digital's 64-bit RISC architecture. EmulatR models the 21264 (EV6) generation.

21264 / EV6 (EV67) -- the Alpha CPU implementation EmulatR emulates; EV67 is the later 21264A.

21272 / Tsunami / Typhoon -- the core-logic chipset: Cchip (control), Dchip (data), Pchip (PCI).

Cchip / Dchip / Pchip -- the three 21272 components: system control and memory arbitration, the data path, and the PCI host bridge.

CY82C693 -- the Cypress PCI-to-ISA southbridge (also the IDE / ATAPI controller and SuperIO).

SRM -- the Alpha console firmware (the >>> prompt) that EmulatR runs unmodified.

PALcode -- Privileged Architecture Library code: the low-level routines (OpenVMS and Tru64 personalities) that implement privileged operations.

SROM -- the serial ROM whose parameters seed the CPU at reset (passed in registers R16-R21).

LFU / UPD> -- the Loadable Firmware Update utility and its prompt, used to update firmware in flash.

NVRAM -- the non-volatile store (backed by the flash image) holding SRM environment variables.

IPR -- Internal Processor Register, accessed in PAL mode via HW_MTPR / HW_MFPR.

ITB / DTB -- the instruction and data Translation Buffers (the EV6 TLBs).

TLB -- Translation Lookaside Buffer: the cache of virtual-to-physical mappings.

SPAM -- Set-Partitioned Associative Memory: EmulatR's sharded, set-associative TLB model.

kseg -- the EV6 superpage region where a virtual address maps directly to physical (no page table).

ASN -- Address Space Number, the tag distinguishing per-process translations.

Box -- a functional execution domain in EmulatR: IBox (control flow), EBox (integer), FBox (floating point), MBox (memory), CBox (cache / predict), PalBox (privileged).

grain -- a single per-opcode leaf function that implements one instruction's semantics.

GCT / FRU -- the firmware's Configuration Tree and Field-Replaceable-Unit data, built during power-up.

HWRPB -- the Hardware Restart Parameter Block, the firmware-to-OS handoff structure.

VMB / SYSBOOT -- the OpenVMS primary and secondary bootstraps; a conversational boot stops at SYSBOOT>.

dqa / dka / dva / ewa -- SRM device names: IDE / ATAPI, SCSI, floppy, and Ethernet respectively.

ATAPI -- the packet command set for optical / removable devices on an IDE channel.

RZ-series -- DEC SCSI disk models (RZ26, RZ28, RZ29, ...).

.axpsnap -- an EmulatR machine-state snapshot file.

NXM -- Non-eXistent Memory: a physical access above the configured RAM limit.

RPCC -- the Alpha cycle-counter instruction (Read Processor Cycle Counter).
