>>> Evaluate Address

At the >>> prompt the SRM console can read and write memory directly -- useful for inspecting device CSRs and firmware state during debugging.

 

Examine (e)

e <space>:<address> reads a location and prints its value. The pmem: space addresses physical memory (including device / chipset MMIO):

>>> e pmem:80130000040

pmem: 80130000040 0000000000000000

 

Deposit (d)

d <space>:<address> <value> writes a value to a location:

>>> d pmem:80130000040 1

 

These are the commands behind the halt-gate diagnostics (for example the arming read of pmem:80130000FF8). Be precise with the address and space prefix; an address that is not in an executable region is rejected ("not executable"), and a malformed line produces a syntax error.
